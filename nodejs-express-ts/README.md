# Express.js powered by TypeScript

Run Express.js written in TypeScript

<br />

## How to run:

1. Install dependencies:

```
npm install
```

or

```
yarn
```

2. Build the app:

```
npm run build
```

or

```
yarn run build
```

3. Run the app:

```
npm run dev
```

or

```
yarn run dev
```
