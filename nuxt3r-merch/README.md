# Nuxt 3 - Merch app

A merch app powered by Nuxt3 and JavaScript.

<br />

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```
or
```bash
# npm
npm install
```
or
```bash
# pnpm
pnpm install
```

<br />

## Development Server

Start the development server on http://localhost:3000

```bash
# yarn
yarn run dev
```
or
```bash
# npm
npm run dev
```

<br />

## Production

Build the application for production:

```bash
# yarn
yarn run build
```
or
```bash
# npm
npm run build
```

Locally preview production build:

```bash
# yarn
yarn run preview
```
or
```bash
# npm
npm run preview
```
