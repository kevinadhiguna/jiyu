# vu3ts-portal

A job portal powered by Vue.js v3, TypeScript, and Composition API.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
